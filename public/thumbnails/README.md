# Portfolio Thumbnails

Place your thumbnail images in this folder.

## Naming Convention

Match the filenames in `config/portfolio.json`:
- festival-poster.jpg
- birthday-card.jpg
- wedding-invite.jpg
- instagram-reel.jpg
- business-ad.jpg
- brand-identity.jpg
- event-poster.jpg
- kids-birthday.jpg
- reception-card.jpg
- product-reel.jpg
- social-ad.jpg
- logo-design.jpg

## Image Specifications

- **Format:** JPG or PNG
- **Recommended Size:** 800x1000px (or maintain aspect ratio)
- **File Size:** Keep under 200KB for fast loading
- **Quality:** Compressed but clear thumbnails

## Tips

1. Use lower quality/compressed versions of your work for thumbnails
2. Maintain consistent aspect ratios for better grid layout
3. Add watermarks if needed
4. Use tools like TinyPNG to compress images

## How It Works

1. Add your thumbnail image to this folder
2. Update `config/portfolio.json` with the image path
3. Add your Canva link for users to view full design
4. The portfolio will automatically display your work!
